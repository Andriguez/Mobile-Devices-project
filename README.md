# PROJECT DESCRIPTION:
Welcome to the Task Manager App (Task'it) designed to help you organize and stick to your daily commitments.
This Android application allows users to create, modify, and track their tasks efficiently.

# KEY FEATURES
## Task Addition:
Quick creation of new tasks (via the FAB button)

## Editing & Deletion: 
Updating and removal of existing tasks.

## Task Filtering:
Distinct viewing of completed tasks and to-do tasks.

## Calendar Integration:
Option to add a task as a reminder in the native calendar.

## Display Holidays:
The application's calendar makes a call to the holidays API which allows to highlight the holidays and display the name of the specific holiday once the day selected.

## Localization Language:
The application allows to switch languages between english and french.

## Read me page:
The application shows the information of the readme inside the app.

# TECHNICAL INFORMATION
Platform: Android
Language: Kotlin
Architecture: MVVM
